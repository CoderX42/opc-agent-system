# 产品愿景

> 版本 V 1.0

## 一、为什么要做这款产品

过去二十年的软件主要经历了两个阶段。第一阶段帮助企业记录,比如说 ERP、CRM、OA 等等。第二阶段帮助企业自动执行,比如说 RPA、Workflow、自动化脚本等。现在我们即将进入第三个阶段,软件不再只是记录或执行,而是具备员工的能力,能理解、能规划、能调用工具、能学习、能协作。这就是 Agent 时代。

我们的判断是:每一家企业最终都会拥有自己的数字员工,每一个创业者都会拥有自己的 AI 团队。

## 产品使命

降低 AI 使用门槛,帮助每个人建立自己的数字员工团队,让创业成本持续下降,让创造价值的人拥有更大的生产力。

## 产品愿景

打造全球领先的 Agent Operating System,帮助用户创建、培养、管理、协同、分享交易数字员工,并最终形成一个完整的 AI 企业生态。

## 这个产品不是什么

它不是聊天机器人,也不是 prompt 工具,也不是自动化脚本、workflow 平台、RAG 平台、MCP 平台,这些都只是底层能力。我们交付给用户的是数字员工。

## 核心设计原则

AI 优先、自然语言优先、可视化优先、可组装、人类可控,关键业务支持人工确认,避免高风险自动执行。

## 目标用户

- **第一阶段**:一人公司创业者、自由职业者、开发者、中小企业主
- **第二阶段**:服务型公司
- **第三阶段**:大型企业以及政府组织

## 成功指标

日活跃数字员工数量,而不是注册用户数。

辅助指标包括:

- 每位用户创建的 Agent 数
- Agent 每日完成任务数
- 协同任务的成功率
- 用户与 Agent 的交互时长

## 一句话总结

不是让用户拥有一个 AI,而是让用户拥有一家由数字员工组成的公司。
