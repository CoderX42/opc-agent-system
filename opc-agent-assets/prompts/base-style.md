Pixel-art production style for OPC Agent System.

Reference mood: Stardew Valley, Terraria, ClawLibrary, Habbo Hotel, RPG Maker.

Rules:
- crisp readable pixel art, no blur, no anti-aliased painterly strokes
- cute office-sim tone, productive digital employee fantasy
- clear silhouettes, small-game scale, readable from 100% zoom
- transparent background for sprites and effects
- consistent palette, 16 to 32 color feeling
- no text baked into sprites unless the asset explicitly requests UI text
- clean grid alignment with equal-size cells
